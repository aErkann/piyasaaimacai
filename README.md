## Tuzak Radar fikri

Tuzak Radar, kalabalığın gördüğü şey ile verinin söylediği şey çeliştiğinde bunu yakalar.

Örnek:

- Riskli pump
- Sessiz boğa
- Favori tuzağı
- KG Var tuzağı
- Kadro tuzağı
- Canlı gol tuzağı
- Likidite tuzağı
